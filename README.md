# Prácticas de Redes y Servidores

Repositorio con prácticas realizadas en máquinas virtuales relacionadas con administración de redes y servicios.

- Configuración de Servidor FTP
- Configuración de Servidor de Correo Electrónico
- Administración de redes
- Asignación dinámica de IPs
- Compartición de archivos en red
- Comunicación entre usuarios mediante correo electrónico

## Tecnologías y herramientas

- Máquinas Virtuales
- Redes cliente-servidor
- Protocolos de red
- Configuración de servicios
